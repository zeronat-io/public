# Contributing to ZeroNAT

We are a small team. At this stage we are not accepting public pull requests or
managing a public issue tracker. GitHub Issues are disabled on this repository.

If you have found a bug, have a question, or want to suggest an improvement,
email us: **support@zeronat.io**

For security vulnerabilities specifically, use **security@zeronat.io**. See
[SECURITY.md](SECURITY.md).

---

## What is and is not open source

The Terraform modules, OS configuration, nftables rules, and documentation in
this repository are Apache 2.0 licensed.

The ZeroNAT agent binary is proprietary and distributed via AWS Marketplace
only. Its source is not published here.
